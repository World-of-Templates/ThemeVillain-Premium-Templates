Cheatsheet

Want to know what class does what? Here you can see a list of all helper (and handy) classes packed with Titan.

Helper classes

CTR+F (WIN) / CMD+F (MAC) this page if you are looking for a specific class.

.vbottom Aligns elemts to the bottom. Used on hero elements, see the index.html for and example.
.text-center Centers the text horizontally.
.white-bg Gives an element a white background
.m-[0,xs,sm,md,lg,xl] Sets margins of an element to a specific value. -0 = 0, -xs = 5px, -sm = 10px and so on.
.mt-[0,xs,sm,md,lg,xl] Sets margin-top of an element to a specific value. -0 = 0, -xs = 5px, -sm = 10px and so on.
.mb-[0,xs,sm,md,lg,xl] Sets margin-bottom of an element to a specific value. -0 = 0, -xs = 5px, -sm = 10px and so on.
.mr-[0,xs,sm,md,lg,xl] Sets margin-right of an element to a specific value. -0 = 0, -xs = 5px, -sm = 10px and so on.
.ml-[0,xs,sm,md,lg,xl] Sets margin-left of an element to a specific value. -0 = 0, -xs = 5px, -sm = 10px and so on.
.p-[0,xs,sm,md,lg,xl] Sets padding of an element to a specific value. -0 = 0, -xs = 5px, -sm = 10px and so on.
.pt-[0,xs,sm,md,lg,xl] Sets padding-top of an element to a specific value. -0 = 0, -xs = 5px, -sm = 10px and so on.
.pb-[0,xs,sm,md,lg,xl] Sets padding-bottom of an element to a specific value. -0 = 0, -xs = 5px, -sm = 10px and so on.
.small-text Sets font-size of an element to 14px;
.dark Gives an element a dark background and all containing elements will have white texts and borders.
.gray-bg Use this on the body to give it a gray background, like the start page or blogger example.
.dark-hero Use this on the body to give handle the header logo and menu on a darker background (or hero).
.boxed Use this class on the body to create a boxed layout.

Element classes

.btn Use this on a button to make it look like one. But combine it with .btn-primary or .btn-secondary
.btn-primary Use this to create a primary button (white on black). Always combine this with the .btn class.
.btn-secondary Use this to create a secondary button (bordered). Always combine this with the .btn class.
.hero Create a fullscreen hero
.hero.small Create a small hero
.hero.smaller Create a smaller hero
.hero.tiny Create a tiny hero
.hero.medium Create a medium hero
.behind Put the hero behind the elements that come next. You can use this on any sized hero but not the fullscreen version.
.bg Create a background that's fullscreen and responsive. See the default page (hero) for a working example.
.striped-list Create a striped list, use this on an unordered list element (ul)
.lined-list Create a lined list, use this on an unordered list element (ul)
.social-list Create a social icons list, use this on an unordered list element (ul)
.checklist Create a checklist, use this on an unordered list element (ul)

Icon classes

.fa Use this on every FontAwesome icon class. FontAwesome Cheatsheet
.ion-* Use this on every Ion icon class. Read more about Ionicons
