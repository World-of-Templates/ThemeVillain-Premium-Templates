The portfolio

This page will guide you through the portfolio feature.

HTML Structure

For any portfolio grid you add, the HTML will look something like this:

<div class="item-grid">
  // The portfolio item
  <a class="item">
    // The Thumbnail
    <div class="thumb">lt;/div>
      <img src="path/to/img" alt="Alt Text"/>
    // The caption
    <div class="caption">
      <h3>The Title</div>
      <div class="tags"> 
        <div class="tag">Tag</div>
      </div>
    </div>
  </a>   
  // A second item
  <a class="item" ...
  // Item sizer. Removing this will break the layout!
  <div class="item-sizer"></div>
</div>

Grid types

Use classes on the ".item-grid" container like ".col-2, .col-3, .col-4" to create columns in the grid.

Masonry

You can also use Masonry if you like, just add the ".masonry" class to the ".item-grid" container.

No spacing

For this, just add the ".no-spacing" class to the container that holds the images.

Item filtering

Review one of the portfolio pages and look for the ".filter-container" div. Every filter that goes into it, has a "data-toggle" attribute. Within this attribute, there is a classname that corresponds with the classname on the element, within the item grid.
