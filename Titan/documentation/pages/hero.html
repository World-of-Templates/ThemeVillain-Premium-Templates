Hero section

This page will explain the hero section.

HTML Structure

For any hero grid you add, the HTML will look something like this:

<section class="hero">
  // The background
  <div class="bg" style="background-image: url(path/to/image);"></div>
  // The bottom aligned container, using vbottom
  <div class="vbottom">
    // The container that has specific dimensions
    <div class="container">
      <div class="row">
        // The content of your hero, in this case within a col-md-6 (half sized) holder
        <div class="col-md-6"> 
          <h1> The title </h1>
          <p> The description </p>
        </div>
      </div>
    </div>
  </div>  
</section>

Dark hero

To make use of a dark hero with light text, simply add a class named ".dark" to the ".hero" class. Becoming <section class="hero dark">

Faded background

When you have a background image that will make the text hard to read, it's often good to add the ".faded" class to the background, like this:
<div class="bg faded" style="background-image: url(path/to/image);"></div>
This will fade your background and the text will be more readable.

Centered text

You can use the ".text-center" class from Bootstrap to have the text centered inside the hero. This class can be added to the .container.

Fullscreen hero

The hero is fullscreen by default. If you'd like to have a smaller hero, then use ".medium", ".small", ".smaller" or "tiny" as an addition on the hero class.
