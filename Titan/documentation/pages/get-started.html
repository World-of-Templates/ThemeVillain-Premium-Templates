Get started

Want to get started with Titan? Please take a moment to read through this documentation that will help you get started.

Pro tips

Whether you are a newbie or pro, these tips might come in handy.

  • Never start from scratch.
  • Use the custom.css included to make changes to your styles.
  • Read the docs carefully before contacting support. You might be helped more quickly.
  • If the docs don't help, use Google or stackoverflow for your specific problem.
  • Use find and replace all, if you'd like to change more files at once.
  • Titan is based on Bootstrap v3.3.6. So you can use all it's handy features.
