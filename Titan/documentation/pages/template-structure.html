Template structure

This page will explain the structure of the template.

The structure

  • Titan
    • /css/ (the CSS styles)
    • /fonts/ (the included fonts)
    • /images/
    • /js/ (the javascripts)
    • /modules/ (modules like the Titan Slider)
    • /pages/ (all the predefined pages)

CSS

  • /css/
    • base.css - All template styles compiled for optimized SEO
    • core.css - The core of Titan, including Bootstrap
    • custom.css - Please use this stylesheet to make modifications

Javascript

  • /js/
    • /assets/ - All the assets needed to load Titan
    • functions.js - The template functions

Modules

  • /modules/
    • /tera-lightbox/ - Tera Lightbox
    • /titan-slider/ - Titan Slider

Pages

  • /pages/

Pages

  • /pages
    • /blog/ - Blog pages
    • /content/ - Content pages
    • /demos/ - Demo's
    • /features/ - Template features
    • /portfolio/ - Portfolio pages
    • /shop/ - Shop pages
