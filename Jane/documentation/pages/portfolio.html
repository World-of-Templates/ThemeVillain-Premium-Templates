A look inside.

The grid

The grid container is a div element that always holds the 'grid' class, and can be combined with a Bootstrap row. 

// The grid container
<div class="row grid">
	// The item
	<a class="item" href="path/to/work-page.html">
		// Item content
	</a>
	<a class="item" href="path/to/work-page.html">
		// Item content
	</a>
	<a class="item" href="path/to/work-page.html">
		// Item content
	</a>
	<a class="item" href="path/to/work-page.html">
		// Item content
	</a>
	... Etc.
</div>

Items

Items are links that include the ".item" class. Within the link there's a thumbnail div element containing the thumbnail image and a caption div for titles and tags.

// The item
<a class="item artdirection" href="path/to/work-page.html"> // Use the "artdirection" class for filtering
	<div class="thumb"> // Thumbnail holder
		<img src="path/to/thumbnail-image.jpg"/> // Thumbnail container
	<div class="caption">
		<h4>Caption title</h4> // Caption title
		<div class="tag">Tag & Tag</div> // Tags or description
	</div>
</a>
// Another item

Filter options

It's easy as adding a link inside the filter container, using a "data-filter" attribute that corresponds with the class that needs to be filtered. You can then use this class on every ".item"

// The Filter container
<div class="filters"> // The filter list
	<li>
		// You can select all items by simply filtering on ".item"
		<a class='filter' href='#' data-filter='item'>All</a>
	</li>
	<li>
		// This filter corresponds with items that have a ".design" class
		<a class='filter' href='#' data-filter='design'>Design</a>
	</li>
	// Etc.
