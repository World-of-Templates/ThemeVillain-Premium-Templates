Getting started

Thank you so much for purchasing Sir!
We hope you will enjoy it.

Sir uses the Bootstrap 3.1.1 framework, which means you can use it's extended library of components, aswell as the column structure.
To get started, please navigate using the buttons above.

The navigation

To make the navigation work, you will at least need the following code for the menu: 

<nav id="main" data-spy="affix">
	<ul>
		<li>
			<a href="#sectionID" class="sectionID>Title</a>
		</li>
	</ul>	
</nav>

And a section with the corresponding ID: 

<section id="sectionID">
// Your content here
</section>

As you might have noticed, we use the same ID three times. Make sure this same ID is used for the link's href and class. Also, to trigger the smooth scroll function, use this ID again as the section's ID.

Make sure you only use this ID for one section.

To set the active menu item after the page is loaded, simply add the .active class to the main menu's link.

How to use the lightbox

It's quite simple! First, make sure you are inside section#portfolio, then put an .item inside it, like: 

<a href="path/to/lightbox/image.jpg" class="item">

Inside the .item link, you can put the thumbnail, which will be clickable and will show the lightbox with the image from the specified url.

How to add video's to the lightbox

For video's to show inside the lightbox, you will need the ID of either a YouTube or Vimeo video link.

Retrieving ID's

For YouTube links, you can find the ID inside the URL right after www.youtube.com/watch?v=

In case you want to show a Vimeo video, use the id located after: http://vimeo.com/ (which is usually a large number).

Implementing

Now, find the <a class='item'> element. And add the following attributes:

data-height="[height]", data-width="[width]", data-type="[vimeo OR youtube]" and data-video="[videoID]".

Use the previously located video id for the last attribute. Your video item can look like:

<a class='item' data-height='700' data-type='vimeo' data-video='21604065' data-width='1000' href='#'>

PHP contact form

First of all; make sure you run your website on a PHP-enabled server. When all the template files are uploaded to the webserver, the PHP contactform works out-of-the-box. The only thing you will need to change is the receivers' e-mail address.
To do that, find the following line inside the email.php file from the php directory of your template folder: 

$to         = 'email@example.com'; //Your e-mail address

Found it? Great! Now the only thing you need to do is change the address between the single quotes.
